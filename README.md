# 💰 PocketWise

 Smart Expense Tracking Made Simple

PocketWise is a beginner-friendly Python project that helps users track their monthly income, monitor expenses, calculate savings, and analyze spending habits.

---

## ✨ Features

✅ Track monthly income

✅ Record expenses in multiple categories:

* Food
* Travel
* Shopping
* Miscellaneous
* Others

✅ Calculate total expenses

✅ Calculate remaining balance

✅ Calculate savings percentage

✅ Warn users when expenses exceed income

✅ Show personalized savings feedback

✅ Display expense breakdown by category

✅ Identify the highest spending category

---

## 📊 Sample Analysis

PocketWise can generate insights such as:

* Savings Percentage
* Expense Breakdown
* Overspending Warnings
* Highest Expense Category
* Financial Suggestions
